# sustechthesis

# Usage
Please read the [document](https://github.com/Iydon/sustechthesis/blob/master/main.pdf).

# Build

* Use Makefile: `make`, `make clean` ......
* Or, use `latexmk`
  * Build: `latexmk main`
  * Clean: `latexmk -c main`
